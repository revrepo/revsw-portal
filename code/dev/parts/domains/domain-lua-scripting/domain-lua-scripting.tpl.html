The Lua scripting functionality allows you to code custom Lua snippets to be executed on edge or origin proxy servers.

  • The Lua support is implemented using Nginx module “lua-nginx-module” - please see https://github.com/openresty/lua-nginx-module for additional details
  • Please note that the Lua code is manually reviewed and approved by customer support engineers before deploying it in the global network. Engineers are notified automatically about submitted Lua code changes so there is no need to open a separate support ticket asking to approve the code.
